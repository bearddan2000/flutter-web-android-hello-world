# flutter-web-android-hello-world

## Description
A POC project for flutter.

## Tech stack
- flutter
- android

## Docker stack
- matspfeiffer/flutter:beta

## To run
`sudo ./install.sh -u`
- Available at http://localhost

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
